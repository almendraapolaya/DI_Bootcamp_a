# Excel Data Analysis Project - Week 7 - Day 2

This repository contains a comprehensive series of data analysis exercises focusing on technical proficiency, statistical modeling, and interactive reporting using Microsoft Excel.

## 📊 Technical Exercises

### Exercise 1: Advanced Logical Functions
- **Implementation:** Developed complex formulas using `IF` and `Nested IF` statements.
- **Key Skill:** Utilized absolute cell references (`$`) to ensure data scaling integrity across large datasets.

### Exercise 2: Data Summarization & Drill-Down
- **Implementation:** Engineered Pivot Tables to filter holiday pricing data.
- **Key Skill:** Applied advanced filters (Label Filters) and utilized the **Drill-Down** feature to verify the granularity of summarized data.

### Exercise 3: Interactive Dashboards & Visualization
- **Implementation:** Created a Pivot Chart visualizing average Oscar nominations categorized by film certificates.
- **Key Skill:** Integrated **Slicers** to create a user-friendly, interactive dashboard experience.

### Exercise 4: Real Estate Data Modeling
- **Implementation:** Created a multi-layered property analysis tool.
- **Key Skill:** 
  - Engineered **Helper Columns** using `IF(OR())` logic to categorize settings.
  - Calculated custom metrics ("Total Rooms") to enhance raw data.
  - Performed **Percentage of Row Total** calculations to analyze market distribution.

### Exercise 5: Multi-Source Report Connections
- **Implementation:** Connected a single Slicer to multiple Pivot Tables across different worksheets.
- **Key Skill:** Managed **Report Connections** to build synchronized data views for Australasian film statistics.

## 🛠️ Core Competencies Demonstrated
- **Data Engineering:** Data cleaning, helper column creation, and custom calculations.
- **Analytics:** Statistical summarization (`AVERAGE`, `COUNT`, `SUM`) and percentage distribution.
- **Visualization:** Pivot Charts and interactive UI elements (Slicers).
- **Logic:** Advanced conditional formatting and formula architecture.
