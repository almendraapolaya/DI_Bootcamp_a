# Daily Challenge: Interactive Data Visualization

##  Project Goal
This project focuses on using **Matplotlib** and **Seaborn** to analyze the US Superstore dataset. The objective was to perform data cleaning and create visualizations that reveal trends in sales, geography, and profitability.

##  Key Insights & Analysis
* **Yearly Growth:** Sales have shown a consistent upward trend from 2014 to 2017, indicating successful business expansion.
* **The Profitability "Red Line":** My Seaborn scatter plot analysis reveals that discounts greater than **20%** significantly increase the risk of negative profit (losses). 
* **Market Leaders:** California and New York are the top revenue-contributing states, representing the most critical markets for the Superstore.

## 🛠️ Skills & Tools Demonstrated
* **Data Cleaning:** Handled Excel date formats and null values using Pandas.
* **Interactive Plotting:** Used Matplotlib to build dynamic trend lines.
* **Statistical Visualization:** Used Seaborn to create elegant bar charts and correlation scatter plots.

## 📁 Files
* `US_Superstore_Analysis.ipynb` - The complete Python code and notebook.
* `US_Superstore_data.xls` - The raw data used for the analysis.
